# 🐍 Snake Game - Clássico Reimaginado

Este é um projeto de um jogo da cobrinha (Snake) moderno e personalizável, focado em uma experiência visual limpa e jogabilidade fluida.

## 👤 Desenvolvedor
Este projeto foi orgulhosamente desenvolvido por **Israel Silva**.

---

## 🚀 Funcionalidades
- **Visual Premium:** Fundo em padrão xadrez (checkerboard) e efeitos de brilho (glow) na comida.
- **Sistema de Recorde:** O jogo salva automaticamente sua maior pontuação no navegador (LocalStorage).
- **Customização de Skins:** Área dedicada no código para carregar imagens personalizadas para a cabeça, corpo e comida.
- **Interface Intuitiva:** Telas de "Início" e "Game Over" integradas conforme os padrões modernos de UI.

## 🕹️ Como Jogar
1. Abra o arquivo `index.html` em qualquer navegador.
2. Clique em **INICIAR JOGO**.
3. Use as **setas do teclado** ou as teclas **WASD** para controlar a direção.
4. O objetivo é comer a fruta para crescer e bater recordes sem colidir com as paredes ou com o próprio corpo.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Canvas para renderização gráfica)
- **CSS3** (Layout responsivo e estilização de overlays)
- **JavaScript** (Lógica de colisão, persistência de dados e motor de jogo)
